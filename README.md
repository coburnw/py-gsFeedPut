# py-gsFeedPut
A Grovestreams Feed-Put library in Python

A limited feature data feeder for streaming data to 
Grovestreams data store using their Feed-Put protocol.

The Grovestreams API-KEY is stored in the users ~/.netrc file
as the password in the project_id stanza: 

```
# ~/.netrc
machine simple
login none
account none
password ******
```
